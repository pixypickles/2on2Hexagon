# Hexa Shoot Arena

六角形の壁付きコートで戦う、2対2のスマートフォン向けアーケードスポーツゲームです。

- 1チーム：キーパー1人＋フィールドプレーヤー1人
- 選手はセンターラインを越えられない
- ボールアウトなし
- 壁反射あり
- 試合時間は1分または2分
- 得点はすべて1点
- 必殺シュートは試合前に1種類選択

## 想定環境

- Android
- 横画面固定
- 2Dトップビュー
- 60 FPS目標
- タッチ操作
- セーブデータ、課金、育成、アンロック要素なし

## 操作概要

画面左に方向入力、画面右にA・B・C・Dボタンを配置します。

### 攻撃時

| ボタン | 動作 |
|---|---|
| A | 速いゴロパス。押した瞬間に操作キャラクター交代 |
| B | ゆっくりした浮き玉パス。押した瞬間に操作キャラクター交代 |
| C | ストレートシュート。長押しで速度上昇。最大チャージでブレ球 |
| D | カーブシュート。長押しで速度と回転量が上昇 |

### 守備時

| ボタン | 動作 |
|---|---|
| A | 操作キャラクター交代 |
| B | 撃ち返し狙い |
| C | 操作対象に関係なくキーパーが長距離パンチングジャンプ |
| D | 操作対象に関係なくキーパーが短距離キャッチジャンプ |

キーパーは立っている状態でもジャンプ中でも、ボールに触れる瞬間にCまたはDを押すとジャストキャッチできます。

## チーム

全チーム同一性能で、違いは名前とユニフォームのみです。

1. **BLIZZARD FOX**：白黒縦縞、水色の差し色、黒パンツ
2. **SALVIDA A**：エンジ色、白パンツ
3. **SALVIDA B**：ターコイズ、白パンツ
4. **TAKE-ZO**：ピンクとネイビーの太いボーダー、白パンツ
5. **漫チェスターP**：全身濃いネイビー

## 開発方針

機能追加より、まず最後まで遊べる完成品を優先します。初期版ではオンライン対戦、育成、ガチャ、能力差、ファウル、オフサイド、スローイン、コーナーキック、PKを実装しません。

## ブラウザ版の起動

このリポジトリには、そのまま遊べるブラウザ版MVPが含まれています。

- `index.html`
- `style.css`
- `game.js`

ローカルでは `index.html` をブラウザで開いてください。GitHub Pagesを使う場合は、リポジトリの Settings → Pages から main ブランチのルートを公開します。

### 現在の実装範囲

- 1P VS CPU
- キーボードによる2P対戦
- 1分・2分
- EASY / NORMAL / HARD
- 5チーム
- 壁反射、パス、浮き球、チャージ、カーブ、ブレ球
- パンチング、キャッチ、撃ち返し、ジャストキャッチ
- チーム共有必殺ゲージ
- 必殺シュート4種類

初期版は外部画像や音声を使わず、Canvas描画だけで動作します。
